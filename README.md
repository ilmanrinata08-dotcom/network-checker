## 📌 Tentang Project

Project ini merupakan tool berbasis web untuk melakukan
pengecekan dan monitoring jaringan secara sederhana.
Seluruh proses pengembangan dilakukan menggunakan
smartphone Android melalui aplikasi Termux — tanpa laptop.

Project ini dibuat sebagai bagian dari proses belajar
mandiri di bidang jaringan komputer dan pemrograman
selama menempuh pendidikan di SMK jurusan TKJ.

---

## ✨ Fitur Utama

- Pengecekan status koneksi jaringan
- Antarmuka web yang dapat diakses melalui browser
- Ringan dan mudah dijalankan
- Dikembangkan sepenuhnya di lingkungan Linux (Termux)
